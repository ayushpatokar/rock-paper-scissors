# Rock Paper Scissors Game 

A simple Rock Paper Scissors game built using **HTML, CSS, and JavaScript**. Play against the computer and see who wins!

## Features

* Interactive user interface
* Random computer choice generation
* Score tracking for player and computer
* Instant result display
* beginner-friendly design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How to Play

1. Choose **Rock**, **Paper**, or **Scissors**.
2. The computer will randomly make its choice.
3. The winner is determined using the standard rules:

   * Rock beats Scissors
   * Scissors beats Paper
   * Paper beats Rock
4. Scores are updated after each round.
